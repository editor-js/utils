# @editorjs/ui-kit

Some UI components for Editor.js

### Installation
 ```
   npm install @editorjs/ui-kit
```

### Components list
- [Popover](./src/popover) - A tool-tip like popover list element with various options for content and position, search input and nested popovers.
- [ScrollLock](./src/scrollLock) - A utility for locking scroll on the page. Useful for popovers, modals, etc.
